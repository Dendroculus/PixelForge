# Code of Conduct

## Our Pledge

We want PixelForge to be a welcoming, respectful, and productive open-source project for everyone.

Everyone participating in this project is expected to act with kindness, professionalism, and respect, regardless of background, experience level, identity, location, or role in the community.

## Expected Behavior

Examples of positive behavior include:

- Using welcoming and respectful language
- Giving constructive feedback
- Being patient with beginners and contributors who are still learning
- Keeping discussions focused on the project
- Respecting different opinions and technical approaches
- Accepting feedback gracefully
- Helping maintain a collaborative and safe environment

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment, insults, discrimination, or personal attacks
- Trolling, repeated disruption, or intentionally derailing discussions
- Publishing private information without permission
- Sexualized language or unwanted attention
- Threats, intimidation, or hostile behavior
- Dismissing contributors based on identity, background, or experience level
- Any behavior that would reasonably make others feel unsafe or unwelcome

## Project Standards

When contributing to PixelForge:

- Keep technical discussions respectful and focused.
- Critique ideas and code, not people.
- Assume good intent when possible.
- Avoid pushing unrelated changes into focused pull requests.
- Respect maintainers' decisions about project direction and scope.

## Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please contact the project maintainer privately.

Please include:

- A description of what happened
- Where it happened, such as an issue, pull request, discussion, or message
- Any relevant links or screenshots
- Whether you would prefer to stay anonymous

Reports will be reviewed with care and handled as privately as possible.

## Enforcement

Maintainers may take appropriate action when this Code of Conduct is violated, including:

- A private warning
- Requesting edits or removal of harmful content
- Temporary restriction from project participation
- Permanent restriction from project participation

The goal is to protect the project community and keep PixelForge a safe place to collaborate.

## Scope

This Code of Conduct applies to:

- GitHub issues
- Pull requests
- Discussions
- Code review
- Project documentation
- Any official PixelForge community spaces

## Maintainer Responsibility

Maintainers are expected to apply this Code of Conduct fairly and consistently. They should also model the respectful behavior expected from the wider community.

Thank you for helping keep PixelForge welcoming and professional.
